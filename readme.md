# 🚀 Employee Performance Predictor using AI & Data Analytics

## 📌 Project Overview

This project is an **AI-powered Employee Performance Prediction System** that uses Machine Learning to predict whether an employee is likely to be a **High/Medium performer or Low performer**.

It also provides:

* 📊 Feature Importance Insights
* 🔍 Explainable AI (SHAP)
* 💡 HR Recommendations for improvement

---

## 🎯 Problem Statement

Organizations often struggle to evaluate employee performance objectively. Traditional evaluation methods can be biased and inefficient.

This project solves that by:

* Predicting employee performance using data
* Identifying key factors affecting performance
* Providing actionable HR insights

---

## 💼 Business Use Cases

* 📈 Identify high-performing employees
* ⚠️ Detect low-performing employees early
* 🎯 Improve promotion decisions
* 🎓 Plan training programs
* 🔁 Enhance employee retention

---

## 🧠 Tech Stack

* Python 🐍
* Pandas, NumPy
* Scikit-learn (Random Forest)
* Streamlit (Dashboard UI)
* Plotly (Visualization)
* SHAP (Explainable AI)

## ⚙️ Installation Guide

### 1️⃣ Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/employee-performance-predictor-ml.git
cd employee-performance-predictor-ml
```

### 2️⃣ Create virtual environment

```bash
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ How to Run

### Step 1: Train the Model

```bash
python main.py
```

### Step 2: Run Dashboard

```bash
streamlit run dashboard/app.py
```
